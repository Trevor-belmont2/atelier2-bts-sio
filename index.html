Application PHP (MVC) – Gestion du personnel

Cette application PHP a été développée pour interagir avec la base de données MediaTek86 modélisée dans Looping et créée via SQL. Elle est structurée selon l'architecture MVC (Modèle - Vue - Contrôleur) et permet d'afficher dynamiquement la liste des personnels enregistrés.

Le code source est disponible dans ce dossier GitHub :

  • 📁 Dossier 05_Application (code source complet de l’application)

L’application est à exécuter en local (via WampServer ou Xampp), depuis le fichier index.php.
